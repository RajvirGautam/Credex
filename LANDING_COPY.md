## FAQ (5)

1. **What's the catch?**
   None. No login, no email required to see results. We do offer to email a copy because it's useful, but the page renders first.
2. **Will you sell my data?**
   No. The shareable URL strips identifying details. We store an audit so the share link works.
3. **How current is the pricing?**
   We pull every number from the vendor's own pricing page and re-verify weekly. The pull date is stamped in PRICING_DATA.md.
4. **Is the engine using AI to make recommendations?**
   No. Recommendations are deterministic, hard-coded rules. AI only writes the optional ~100-word personalized summary, where a wrong number doesn't matter.
5. **What if you can't help me?**
   Then you'll get a "you're spending well" page. We won't invent a problem to manufacture a CTA.

## Voice rules (used throughout the site)

- No exclamation points.
- No emojis.
- "You're spending well" is a valid outcome — never an upsell.
- Always reference the real number, never a marketing range ("up to 40% savings" is banned).
- Citations are always one click away.
